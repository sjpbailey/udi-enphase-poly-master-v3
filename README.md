* Node Server for Enphase solar inverter system.

## Installation

### Node Settings

The settings for this node are:

#### Short Poll

* Every 10 Minutes

#### Long Poll

* Not used

#### nodes

* How many child nodes to create

#### multiplier

* Apply the multiplier to count and save in GV1

## Requirements

1. Polyglot V3.
2. ISY firmware 5.3.x or later
